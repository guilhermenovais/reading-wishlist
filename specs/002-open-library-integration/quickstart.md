# Quickstart: Open Library Integration

**Feature**: 002-open-library-integration | **Date**: 2026-06-17

## Prerequisites

- Node.js 18+
- PostgreSQL running locally
- Feature 001 (Reading Wishlist MVP) fully implemented and working

## Setup

1. **Install dependencies** (if new packages were added):
   ```bash
   npm install
   ```

2. **Run the new database migration** (adds isbn and publicationYear columns):
   ```bash
   npx prisma migrate dev
   ```

3. **Generate Prisma client** (if not auto-generated by migrate):
   ```bash
   npx prisma generate
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

## Verify

1. **Search for books**: Navigate to `http://localhost:3000`, click the search link, enter a book title, and verify results appear from Open Library.

2. **Import a book**: Click the import button on a search result. Verify the book appears in the wishlist with ISBN and publication year.

3. **View imported book details**: Click on the imported book in the wishlist. Verify ISBN and publication year are displayed.

4. **Duplicate detection**: Try importing the same book again (same ISBN). Verify the system prevents the duplicate.
